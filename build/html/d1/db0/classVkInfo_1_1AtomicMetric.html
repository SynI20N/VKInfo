Detailed Description

template<typename T>
class VkInfo::AtomicMetric< T >

A thread-safe metric implementation using atomic variables.

AtomicMetric represents a numeric metric (e.g., CPU usage, request count) that can be safely updated and accessed across multiple threads. It supports setting values directly or assigning a random value within a defined range. Upon retrieval via get_value_and_reset(), the value is reset.

Template Parameters
TThe numeric type of the metric (e.g., int, float, double).
