\doxysubsection{Detailed Description}
Periodically logs metric values to a file. 

\mbox{\hyperlink{classVkInfo_1_1MetricsLogger}{Metrics\+Logger}} runs in a background thread, collecting metric values from a \mbox{\hyperlink{classVkInfo_1_1MetricsRegistry}{Metrics\+Registry}} at fixed intervals and writing them to a log file with timestamps. After logging, each metric is reset. This logger is thread-\/safe and non-\/blocking for metric producers.
