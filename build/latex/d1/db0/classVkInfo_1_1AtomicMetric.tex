\subsubsection*{template$<$typename T$>$\newline
class Vk\+Info\+::\+Atomic\+Metric$<$ T $>$}
A thread-\/safe metric implementation using atomic variables. 

\mbox{\hyperlink{classVkInfo_1_1AtomicMetric}{Atomic\+Metric}} represents a numeric metric (e.\+g., CPU usage, request count) that can be safely updated and accessed across multiple threads. It supports setting values directly or assigning a random value within a defined range. Upon retrieval via \mbox{\hyperlink{classVkInfo_1_1AtomicMetric_a6ea8de63ea21c4d6a8e3c30838e381db}{get\+\_\+value\+\_\+and\+\_\+reset()}}, the value is reset.
